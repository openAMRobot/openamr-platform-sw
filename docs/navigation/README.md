# docs/navigation

Top-level pointer for Nav2 documentation.

The Nav2 bringup + SLAM + RViz nav layout live in:

**`ros2/src/openamrobot_nav2/`**

See its [README](../../ros2/src/openamrobot_nav2/README.md) for the parameter highlights and the recommended composition pattern.

Tuning notes specific to the docking scenario (e.g. why `velocity_smoother.max_decel` is softened, why `collision_monitor.FootprintApproach` is disabled) are recorded in the docking package's [`docs/12_lessons_learned.md`](../../ros2/src/openamrobot_docking/docs/12_lessons_learned.md).
